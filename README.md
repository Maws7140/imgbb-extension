# ImgBB Quick Upload

Simple extension to right-click upload images to ImgBB.


## Install

| Browser | Link |
|---------|------|
| Chrome | [Chrome Web Store](https://chromewebstore.google.com/detail/imgbb-quick-upload/bnlhhjlcnmokpcimicmnnmebakhlagbn) |
| Edge | [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/imgbb-quick-upload/bjdmpbiblpmnhgohkelnjkokbenghohj) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/imgbb-quick-upload/) |

## Features
- Auto-copy link or BB code
- Auto-delete timer
- Local history
